# API Discovery

A curated marketplace of free APIs for developers. Browse and discover APIs for your projects.

## Features

- Searchable API database
- Category filtering
- Authentication type display
- Rate limit information
- HTTPS status
- User ratings

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000 to browse APIs.
